# spring-cloud：what 开源分布式 架构。
我司主要使用了他的几个主要的组件，包括
服务: springboot  [https://spring.io/projects/spring-boot](https://spring.io/projects/spring-boot)
网关: zuul
服务发现: eureka
rpc: feign(基于http)
负载均衡: robbin(客户端实现)
链路追踪: sleuth


消息队列: kafka
缓存： redis
